API Documentation

These are the list of APIs you can use to access, add, update, and delete from the database. See this page on Postman for the detailed documentation.

All Cafes

Method Description
GET Search cafes by location. The /api/search route will search the cafe database for a cafe that matches the location. Use the location parameter to pass a location name.
GET Get all cafes. The /api/all route returns a JSON with the data from all cafes in the database.
POST Add a new cafe. The /api/add route lets you add a new cafe into the database. Enter all the required parameters in the body field.
PATCH Update the price of a black coffee at a particular cafe using the /api/update_price/ route. You need to use the id and new_price parameters.
DELETE Delete a cafe by Id. Deletes a cafe from the database using the api/report-closed route. You will need to provide the Id of the cafe to delete as a route. You will also need to provide a valid API key for this operation to be allowed.
